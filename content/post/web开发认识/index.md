# 小白入门

* 前后端分离
* 前端是视图，需要数据去后端
* 后端是数据，只返回数据（大部分是JSON） 

前端框架： Vue,React

后端框架：Speingboat,Express 
## 用户和服务器交互是API接口传输JSON数据
http 流程：

1：用户向后端发送窗口（request）

2:后端处理数据（计算，数据库交互，其他的一些逻辑

3.后端返回一个（response）

API：应用编程接口
